ROV

Remotely Operated Underwater Vehicle

The ROV

The ROV is a small underwater system that is equipped with 6 thrusters that enable it to control all 6 degrees of freedom under water. It uses an inertial measuerement unit (IMU), a camera and a pressure sensor to estimate it's position, velocity, orientation and angular velocities about all three body-fix axes. The ROV is remotely operated via a PC using a Xbox controller or by setting references in specific states through the GUI. The PC communicates with the ROV's internal computer through a cable.
One of the main purposes of this project has been to introduce a positioning system in order to be able to fulfill the ultimate goal of a fully autonomous system in the future. This goal has been achieved with the position control, which enables the operator to specify a position to which the ROV then navigates.
